# THWS Bachelor Thesis Guides

This repository provides two public guidance documents for students preparing a bachelor thesis.

The material is currently most directly applicable to students in the Degree Programme Mechatronics (IMC) at the Technische Hochschule Wuerzburg-Schweinfurt (THWS) and related engineering programs. Some administrative links are program-specific and may point to the THWS intranet, where access may require login.

## Contents

- [Bachelor Thesis Guide](guides/bachelor-thesis.md)
- [Bachelor Thesis Writing Guideline](guides/bachelor-thesis-writing-guideline.md)

## How to use this repository

1. Start with the bachelor thesis guide for scope, prerequisites, exposé expectations, structure, and grading criteria.
2. Use the writing guideline while drafting the thesis to make decisions on writing style, figures, equations, notation, citations, and final checks.
3. Treat this repository as practical guidance. Official university regulations, examination rules, and formally issued documents take precedence.

## Scope

This repository is intentionally limited to public-facing guidance documents. 

## Notes

- Some linked resources point to THWS intranet pages or program documents and may require institutional access.
- Program-specific rules should be read together with the current examination regulations and module handbook.
- In case of discrepancies between this repository and official university documents, the latter take precedence.
- The documents in this repository may be expanded later for additional study programs.
